# Portfolio Website (Local Hosting)

This is a static portfolio site with modern CSS and JavaScript interactions.

## Run locally

### Option 1: npm (recommended)

```bash
npm start
```

Then open:

- `http://localhost:4173`

### Option 2: custom port

```bash
PORT=8080 npm start
```

Then open:

- `http://localhost:8080`

## Validation

```bash
npm run check
```
